Descripción:  
Este proyecto corresponde al primer trabajo práctico de la asignatura de Backend.  
Consiste en la implementación de una aplicación web básica utilizando Python.  
El objetivo es aplicar conceptos fundamentales de programación del lado del servidor,  
manejo de rutas, procesamiento de datos y generación dinámica de contenido.  

Tecnologías utilizadas:  
- Python 3  
- HTML básico  

Estructura del proyecto:  
mi_proyect_web/  
├── index.html  
├── main.py  

Instrucciones de uso:  
1. Clonar el repositorio:  
   git clone https://github.com/EELBASCU/Trabajo-1-BackEnd.git  

2. Acceder a la carpeta del proyecto:  
   cd Trabajo-1-BackEnd/mi_proyect_web  

3. Ejecutar el archivo principal:  
   python main.py  

Requisitos:  
- Python 3 instalado  
- (Opcional) Crear entorno virtual:  
   python -m venv env  
   source env/bin/activate (Linux/macOS)  
   .\env\Scripts\activate (Windows)  

- Instalar dependencias (si se incluye requirements.txt):  
   pip install -r requirements.txt
